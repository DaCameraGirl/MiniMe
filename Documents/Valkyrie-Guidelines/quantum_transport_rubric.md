# Quantum Transport Analysis Rubric

## Core Criteria (Weight +10 each)
- Response explicitly compares surface state robustness in Bi₂Se₃ vs. (Bi,Sb)₂Te₃ alloys
- Addresses bulk conductivity tradeoffs when optimizing surface-to-bulk ratio
- Proposes specific heterostructuring approach (e.g. substrate choice, capping layers)
- Quantifies disorder thresholds using Ioffe-Regel criterion or similar metric

## Enhancement Criteria (Weight +5 each)
- Mentions strain engineering effects on Dirac cone topology
- Conserves time reversal symmetry in proposed solutions
- References recent (post-2020) ARPES data on gap stability

## Negative Criteria (Weight -10 each)
- Suggests doping strategies that would collapse bulk gap
- Proposes room-temperature operation as viable
- Uses semiclassical Boltzmann approach without quantum corrections
- Fails to distinguish between trivial and topological surface states

## Validation Checks
1. Optimal solutions must maintain k·p parameters within experimentally achievable ranges (e.g., Fermi velocity ~5e5 m/s)
2. Any proposed superlattice structures should preserve crystalline symmetry
3. Surface state penetration depth must exceed 2nm for proper hybridization control
