# AI Rubric Generator Prompt

## Context
You are building a tool to help educators create assessment rubrics. The system needs to handle multiple criteria types and generate evaluation templates.

## Task
Create a React component that:
1. Allows dynamic addition/removal of rubric criteria
2. Supports different rating scales (numeric, descriptive)
3. Generates printable PDF output
4. Includes example rubrics for common subjects

## Constraints
- Use TypeScript for type safety
- Implement responsive design for mobile/desktop
- Store rubrics in localStorage
- Follow WCAG accessibility standards
